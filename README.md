# Aliyun OSS Upload

Upload files to Aliyun OSS (Object Storage Service) as WordPress media library storage, with enhanced OSS protocol wrapper and full native image editing functions.

## Features

* Remote image auto upload support
* Large file multipart upload support
* Backup mode for easier switching
* Original protocol wrapper supports all native upload functions
* Support for changing default image editor class
* Auto identify and generate WEBP format
* Chinese and English settings with examples

## Download

https://github.com/karrychow/wp-aliyun-oss-upload/archive/main.zip

## Installation

1. Upload the plugin folder to `/wp-content/plugins/` directory of your WordPress site
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings → Aliyun OSS Upload to configure

## About This Fork

This plugin is based on the original work by xiaomac: [https://github.com/xiaomac/oss-upload](https://github.com/xiaomac/oss-upload)

The original author hasn't updated the project for several years and it was no longer maintained. To adapt to new WordPress versions, I've made updates based on the original project. The plugin itself is well-designed, doesn't conflict with other plugins, and uses a very reasonable approach for image display.
