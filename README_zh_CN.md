# Aliyun OSS Upload (中文版)

使用阿里云 OSS (对象存储) 作为 WordPress 媒体库存储，支持增强的 OSS 协议包装和全原生图像编辑功能。

## 功能特性

* 支持远程图片自动上传到 OSS
* 支持大文件自动分块上传
* 支持备份模式，方便在本地和 OSS 之间切换
* 原创增强的协议包装，支持所有 WordPress 原生上传和媒体功能
* 支持自定义图像编辑器类
* 自动识别并生成 WEBP 格式图片
* 完善的设置选项，并附带示例

## 下载

https://github.com/karrychow/wp-aliyun-oss-upload/archive/main.zip

## 安装步骤

1. 将插件文件夹上传到 WordPress 站点的 `/wp-content/plugins/` 目录
2. 在 WordPress 后台的“插件”菜单中启用该插件
3. 前往“设置” → “Aliyun OSS Upload”进行配置

## 关于此分支

本插件基于 xiaomac 的原创作品：[https://github.com/xiaomac/oss-upload](https://github.com/xiaomac/oss-upload)

原作者多年未更新，项目已停止维护。为了适配新版本的 WordPress，我基于原项目进行了更新和维护。该插件设计精良，与其他插件兼容性好，且采用了一种非常合理的图片显示方案。
